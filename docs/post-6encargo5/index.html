Ejercicio de Fotogrametría

Esta es la ciencia de extraer mediciones confiables bidimensionales de imágenes, generalmente fotográficas.

Para este ejercicio se nos pidió explorar el programa Autodesk ReCap, tomándole fotos a un plato de comida creado por nosotros y generar su fotogrametría 3D a partir de sus imágenes.

Ejercicio Fotografía + Origami

Para este ejercicio se nos pidió trbajar con un código de grasshopper creado para generar figuras de origami, luego modelar una que nos inspirara y generar un molde a partir de esta, y finalmente usarlo para insertarle alimento y probar su efectividad.

(Lamentablemente, mi molde y origami terminaron siendo distintos debido a que no logré la forma esperada cuando traté de retratarlo en la vida real, pero de igual manera me gustaron mucho los resultados :))
